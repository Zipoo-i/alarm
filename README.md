# Alarm & Timer System (C / Windows)

Простая система будильников и таймера с хранением в SQLite и планировщиком Windows (schtasks).

## Состав
- `alarm_manager.exe` – консольное меню для управления будильниками и запуска таймера.
- `trigger.exe` – запускается планировщиком при срабатывании будильника (показывает окно и ждёт `stop`).
- `timer.exe` – таймер с обратным отсчётом (секунды) и звуковым сигналом.

## Сборка
Запустите `build.bat` (требуется MinGW gcc). Все исполняемые файлы появятся в `CODE\run`.

## Запуск
- Запустите `CODE\run\alarm_manager.exe` (желательно от имени администратора).
- Выберите пункт 1, введите время (HH:MM), заметку (текст), тип (0 – одноразовый, 1 – ежедневный).
- Для просмотра и удаления используйте пункты 2 и 3.
- Таймер запускается из пункта 4 – укажите время в минутах или формате HH:MM.

## Примечания
- Для работы планировщика необходим запуск от имени администратора.
- Одноразовые будильники автоматически удаляются после срабатывания и ввода `stop` в консоли.
- Заметки отображаются при срабатывании (в окне и консоли).
- Логи `trigger.exe` сохраняются в `trigger_log.txt`.

## Файлы
- `CODE\alarms.db` – база данных (создаётся автоматически).
- Все исполняемые файлы и БД находятся в `CODE\run`.
